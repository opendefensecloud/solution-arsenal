    Documentation Setup¶

    The documentation of the SolAr project is written primarily using Markdown. All documentation related content can be found in https://github.com/opendefensecloud/solution-arsenal/tree/main/docs. New content also should be added there.

    To render the documentation with mkdocs locally you have to

    • once or when you alter the Dockerfile you need to rebuild the custom container image with make docs-docker-build
    • run make docs to render the documentation
