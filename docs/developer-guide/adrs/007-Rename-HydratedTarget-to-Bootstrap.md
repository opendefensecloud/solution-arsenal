# Rename `HydratedTarget` to `Bootstrap`

## Context and Problem Statement

The name `HydratedTarget` is cumbersome and unintuitive.

## Considered Solutions

Rename it to:

- `Bootstrap`
- `EntryPoint`
- `AppOfApps`
- `App`
- `Installation`

## Decision Outcome

After a poll it was decided to rename `HydratedTarget` to `Bootstrap`.
